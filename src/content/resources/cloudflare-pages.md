Cloudflare Pages 适合个人内容站早期部署。连接 Git 仓库后，推送代码即可触发构建和发布。

## 部署要点

- 构建命令：`npm run build`
- 输出目录：`dist`
- 生产域名确定后，设置 `SITE_URL` 环境变量。
